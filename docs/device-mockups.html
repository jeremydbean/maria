Integration Notes

These are product notes for the native app build, not medical claims.

Apple Watch

Use a watchOS companion for one-tap puff logging, urge-only logging, delay timers, haptics, and a complication showing the current vape-free gap.

Widgets

Use WidgetKit for Lock Screen and Home Screen widgets that display the current gap, next goal, today count, and Cash Mode encouragement.

Sync

Sync iPhone, widgets, and Apple Watch through shared local app data and WatchConnectivity so logging from the watch updates the phone timeline.
